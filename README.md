# Toodly

**Chrome Browser Extension**

Ultimate to-do manager built directly into your browser for easy access to tasks

## To-Do
- Set due dates and deadlines
- Mark priority tasks
- Organize tasks into:
  - Today
  - Archive
  - Upcoming

## Timer
- Built-in Pomodoro-style timer
- Simple visual tracker to monitor time progress

## Mosaic View
- View visual progress of completed tasks
- Track daily study and work sessions
